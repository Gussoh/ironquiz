## PLAYING

Run release.bat and follow the instructions on Apache Ant.

Start the server using the "_RUN_SERVER.BAT"-file. Otherwise you won't get a console window.
The other clients you can start by double clicking their respective jar file.

Regarding the Jeopardy music buttons in the admin GUI. You will need to "find" the wav files yourself. 
Save them as sounds/jeopardy_intro.wav, jeopardy_second_half.wav and jeopardy_tada.wav
Connect speakers to the main screen.

Enjoy!


## DEVELOPING
Use netbeans because of the GUI stuff.
Build by calling the ant compile or release targets.
